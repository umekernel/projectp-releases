# projectp-releases

Public distribution repository for ProjectP release assets.

Use GitHub Releases and the stable `releases/latest/download/...` URLs from this repository to distribute launcher packages, update archives, and metadata.

Primary assets:

- `projectp-cli-launcher.zip`
- `client-update.zip`
- `client-version.txt`
- `p-linker-launcher.zip`
- `linker-update.zip`
- `linker-version.txt`
- `projectp-sdk.zip`
- `manifest.json`

This public repository intentionally stays release-facing only. Internal build, packaging, and deployment automation is maintained outside this repo.
